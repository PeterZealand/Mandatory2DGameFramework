\doxysubsection{Detailed Description}
Represents an abstract base class for objects in the game world, providing common properties and behavior. 

The \doxylink{class_mandatory2_d_game_framework_1_1_worlds_1_1_world_object}{World\+Object} class serves as a foundation for all objects that exist within the game world. It includes properties such as Name, Lootable, and Removable to define the object\textquotesingle{}s characteristics. Equality and comparison operations are based on the Name property, allowing objects with the same name to be considered equal.
