Detailed Description

Represents an abstract base class for objects in the game world, providing common properties and behavior.

The WorldObject class serves as a foundation for all objects that exist within the game world. It includes properties such as Name, Lootable, and Removable to define the object's characteristics. Equality and comparison operations are based on the Name property, allowing objects with the same name to be considered equal.
